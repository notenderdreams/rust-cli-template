# Rust CLI Template

[![template](https://img.shields.io/badge/template-cargo--generate-4a90e2)](https://github.com/cargo-generate/cargo-generate)
[![rust](https://img.shields.io/badge/rust-2024-orange)](https://www.rust-lang.org/)
[![license](https://img.shields.io/badge/license-MIT-green)](LICENSE-MIT.txt)

Generate a new project from this template with:

```bash
cargo generate --git https://github.com/notenderdreams/Rust-CLI-Template.git
```

The generated project uses:

- `clap` for command parsing
- `anyhow` and `thiserror` for error handling
- a `hello` subcommand as the starter command
